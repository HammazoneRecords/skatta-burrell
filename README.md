# Skatta Burrell — Official Site

Bespoke artist/industry site built under the **RAAS (Reggae As A Service)** Digital Label Service.

| | |
|---|---|
| **Domain** | `skattaburrell.com` |
| **Subject** | Skatta Burrell — Jamaican producer legend, A&R executive, radio personality |
| **Stack** | Vite + React + TypeScript + Tailwind CSS |
| **Deploy target** | Vercel |
| **License model** | ADTL (Artist Digital Territory License) — site transfers to buyer on purchase |

## Status

🔲 In development — part of the RAAS ADTL portfolio

## Run Locally

```bash
npm install
npm run dev
```

## ADTL Transfer Process

1. GitHub repo transfer to buyer's account
2. Vercel project import from new owner's GitHub
3. Namecheap DNS update — `skattaburrell.com` pointed to new Vercel deployment

All contact/booking forms ship in demo mode until activated by the owner post-transfer.
